# My Portfolio

React를 활용하여 제작한 개인 포트폴리오 웹사이트입니다.  
프로젝트 경험과 기술 스택을 시각적으로 정리하여 소개합니다.

Live Demo: https://portfolio-sandy-delta-83.vercel.app/

---

## Tech Stack

### Frontend
- React
- JavaScript
- CSS

### Tools
- Git
- GitHub
- Vercel

---

## Features

- 프로젝트를 카드 형태로 시각적으로 표현
- 모달을 통한 상세 프로젝트 정보 제공
- 섹션별 스크롤 네비게이션
- 반응형 UI 디자인
- GitHub 및 연락처 링크 연결

- 모달 및 UI 인터랙션 구현
